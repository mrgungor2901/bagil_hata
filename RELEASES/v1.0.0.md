# v1.0.0 — 2026-02-07

Kısaca: İlk resmi sürüm — küçük iyileştirmeler, stil ve deprecation düzeltmeleri.

Detaylar:
- `lib/main.dart` içinde isimlendirme ve deprecated kullanımlar düzeltildi:
  - `Ayarlar.koyu_tema` → `Ayarlar.koyuTema` (okuma/yazma adaptörleri güncellendi)
  - `ColorScheme.background` kullanımına alternatif olarak `surface` kullanımı ayarlandı
  - `.withOpacity()` çağrıları ARGB sabitleriyle değiştirildi ve platform uyumluluğu sağlandı
  - `Slider` / `Switch` renk ayarlarında uyumluluk iyileştirildi

- Basit hata düzeltmeleri ve kod temizliği:
  - Hive adaptörleri doğrulandı
  - `flutter analyze` ve testler çalıştırıldı, sorun kalmadı

Üretim yapıları:
- Android APK: `build/app/outputs/flutter-apk/app-release.apk`

Nasıl ilişkilendirilir (GitHub):
1. Eğer `gh` CLI yüklüyse:

```bash
gh release create v1.0.0 -F RELEASES/v1.0.0.md
```

2. Veya GitHub web arayüzünden yeni bir Release oluşturup tag olarak `v1.0.0` seçin ve bu dosyanın içeriğini açıklama olarak yapıştırın.

Notlar:
- Eğer release notlarında eklemek istediğiniz başka detaylar (sürüm notları, önem derecesi, değişiklik listesi vb.) varsa söyleyin; ben dosyayı günceller ve tekrar pushlarım.
